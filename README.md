## 🛠️ How to Run

1. Ensure Java Development Kit (JDK) is installed:
```bash
java --version
```

2. Navigate to specific problem directory:
```bash
cd "folder_name"
```

3. Compile Java file:
```bash
javac FileName.java
```

4. Run the program:
```bash
java FileName
```
